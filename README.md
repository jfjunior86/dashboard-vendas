# dashboard-vendas
Dashboard com resultado de vendas
O dashdoard apresenta resultados fictícios das vendas mensal, trimestral e anul do XBOX GAME PASS. O objetivo foi transformar dados brutos em informações visuais claras e úteis, o que permite uma análise eficaz do desempenho de vendas e a tomada de decisões baseadas em dados.

Instruções para reprodução.
Baixe o arquivo em anexo e abra usando o excel.
